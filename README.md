# 摄影后期工具箱

本工具箱包含两个实用脚本，旨在提升摄影后期的文件整理与处理效率。

## 1. 自动化照片整理工具

自动扫描指定文件夹，智能筛选出成对的照片文件，并将它们归档到新目录中。

### 核心功能
*   **智能配对**：仅复制**同时存在** `.JPG` 和 `.NEF` (Raw) 格式的照片文件。
    *   *注：如果某张照片只有 JPG 或只有 NEF，将会被自动忽略。*
*   **自动归档**：基于源文件夹名称，自动创建名为 `源文件夹_完成整理` 的新目录。
*   **安全校验**：复制后自动校验文件完整性，防止覆盖同名文件。
*   **日志追踪**：在源文件夹内自动生成详细的操作日志文件。

### 如何使用
1.  双击运行 **`start_organize.bat`**。
2.  根据提示**输入源文件夹路径**（支持直接将文件夹拖入命令行窗口）。
3.  等待脚本执行完毕。

---

## 2. 多图智能拼接工具

交互式图片合并脚本，支持将多张副图按指定方向拼接到主图上。

### 核心功能
*   **智能缩放**：副图会自动按接缝边长进行等比缩放，完美贴合主图。
*   **多图支持**：支持一次性拼接多张副图。
*   **灵活布局**：支持四个方向拼接（上、下、左、右）。
*   **自动命名**：生成文件名为 `主文件名_combined.jpg`，保存在主图同目录下。

### 如何使用
1.  双击运行 **`start_combine.bat`**。
2.  按提示依次输入：
    *   **主图片路径**。
    *   **副图片路径**（输入一张回车一次，直接回车结束输入）。
    *   **拼接方向编号**（1=左, 2=右, 3=上, 4=下）。
3.  脚本将自动生成并保存拼接后的图片。
