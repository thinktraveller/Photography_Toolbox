# 图片合并工具 (Pictures Combination)

一个功能强大的交互式 JPG 图片合并脚本，支持将多张副图片与主图片按指定方向进行智能合并。

## 🌟 功能特性

- **多图片合并**：支持一次性合并多张副图片到主图片
- **智能缩放**：自动按接缝维度等比缩放副图片，确保完美贴合
- **四向合并**：支持左、右、上、下四个方向的图片合并
- **交互式操作**：友好的命令行交互界面，操作简单直观
- **EXIF 兼容**：自动处理图片 EXIF 方向信息
- **高质量输出**：使用 LANCZOS 算法确保缩放质量
- **版本兼容**：兼容不同版本的 Pillow 库
- **持续运行**：脚本持续运行，可连续处理多组图片

## 📋 系统要求

- Python 3.6+
- Pillow 库

## 🚀 安装说明

1. **克隆或下载项目**
   ```bash
   git clone <repository-url>
   cd PicturesCombination
   ```

2. **安装依赖**
   ```bash
   pip install pillow
   ```

## 📖 使用方法

### 基本使用

1. **运行脚本**
   ```bash
   python combine_jpg.py
   ```

2. **按提示操作**
   - 输入主图片路径
   - 逐个输入副图片路径（按 Enter 结束输入）
   - 选择合并方向（1-4）

### 合并方向说明

| 编号 | 方向 | 说明 |
|------|------|------|
| 1 | 左侧 | 副图片依次排列在主图片左侧 |
| 2 | 右侧 | 副图片依次排列在主图片右侧（默认） |
| 3 | 上方 | 副图片依次排列在主图片上方 |
| 4 | 下方 | 副图片依次排列在主图片下方 |

### 排列规则

- **右侧合并**：[主图] → [副图1] → [副图2] → [副图N]
- **左侧合并**：[副图N] → [副图2] → [副图1] → [主图]
- **下方合并**：[主图] ↓ [副图1] ↓ [副图2] ↓ [副图N]
- **上方合并**：[副图N] ↓ [副图2] ↓ [副图1] ↓ [主图]

## 🔧 技术特性

### 图片处理
- **自动格式转换**：所有图片自动转换为 RGB 格式
- **EXIF 处理**：自动应用 EXIF 方向信息，确保图片正确显示
- **高质量缩放**：使用 LANCZOS 重采样算法保证缩放质量

### 智能缩放算法
- **水平合并**：副图片按主图片高度等比缩放
- **垂直合并**：副图片按主图片宽度等比缩放
- **保持比例**：确保副图片不变形，保持原始宽高比

### 输出设置
- **文件命名**：自动生成 `原文件名_combined.jpg` 格式
- **保存位置**：输出到主图片所在目录
- **图片质量**：JPEG 质量设置为 95，启用优化

## ⚠️ 注意事项

1. **文件路径**：请使用完整的文件路径，支持中文路径
2. **图片格式**：脚本主要针对 JPG 格式优化，其他格式会自动转换
3. **内存使用**：处理大图片时请注意内存使用情况
4. **文件覆盖**：如果输出文件已存在，会被自动覆盖

## 🐛 常见问题

### Q: 提示 "未安装 Pillow 库" 怎么办？
A: 运行 `pip install pillow` 安装 Pillow 库。

### Q: 图片合并后质量下降怎么办？
A: 脚本已使用高质量设置（JPEG 质量 95 + LANCZOS 缩放），如需更高质量可考虑使用 PNG 格式。

### Q: 支持其他图片格式吗？
A: 脚本可以读取多种格式（PNG、BMP、TIFF 等），但输出固定为 JPG 格式。

### Q: 如何退出程序？
A: 在任何输入提示时输入 `Exit`（不区分大小写）即可退出。

### Q: 副图片尺寸差异很大怎么办？
A: 脚本会自动按接缝维度等比缩放，确保完美贴合，无需担心尺寸问题。
